# landing-ea

Landing website for an agro consultancy business built as a fast, responsive static site.

## Overview

This repository contains a single-page landing site for NODOAGRO, a consultancy focused on agroeconomic studies and rural subdivision services. The site includes:

- A hero section with a call-to-action
- Service presentation
- About section
- Contact details
- Responsive navigation

## Technologies

- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Montserrat)
- Static image assets

## Files

- `index.html` – main landing page markup
- `style.css` – placeholder for additional custom CSS (currently empty)
- `images/` – image assets used in the site

## Usage

1. Open `index.html` in a browser.
2. Host the folder on any static web server if needed.
   - Example: use VS Code Live Server or serve with Python:
     ```bash
     python3 -m http.server 8000
     ```
3. Visit `http://localhost:8000` after starting the server.

## Customization

- Update the branding, text, and contact details directly in `index.html`.
- Replace the hero background image in the inline CSS inside `index.html`.
- Add custom styles to `style.css` and include the stylesheet link in `index.html` if additional styling is needed.

## Deployment

The site can be deployed to any static hosting provider, such as GitHub Pages, Netlify, or Vercel.

## Contact

For changes or additional content, edit `index.html` with your desired text and assets.
